## Typography

**Font Family:** Hack, monospace

The logo uses a monospace font to reflect the technical/terminal nature of the application.

---

## Usage Guidelines

1. **Minimum Size:** Do not use the full logo smaller than 100px wide
2. **Clear Space:** Maintain padding equal to the height of the "m" around the logo
3. **No Modifications:** Do not alter colors, proportions, or add effects
4. **Background Contrast:** Ensure sufficient contrast between logo and background
5. **Favicon:** Use only the dedicated favicon variant for browser tabs

---

## File Formats

All logos are provided in SVG format for:
- Infinite scalability
- Small file size
- Easy color modifications if needed
- Web and print compatibility

For raster formats (PNG, ICO), convert from SVG at the required resolution.
